# 🚀 CrowdPulse

> 🎯 *Find the fastest route, shortest queue, and safest move — live.*

---

## 📌 Overview

**CrowdPulse** is a real-time crowd intelligence platform designed for large-scale venues like stadiums, concerts, and festivals. It helps attendees navigate efficiently using live congestion data, queue predictions, and smart routing.

It also provides organizers with a powerful dashboard to monitor crowd flow, detect bottlenecks, and send real-time alerts.

---

## ✨ Features

### 👤 Attendee Side
- 📍 Live venue heatmap
- ⏳ Queue time estimation
- 🧭 Smart route suggestions
- 🔔 Real-time alerts & updates
- 👥 Meet-up mode (find friends easily)
- ♿ Accessibility-friendly navigation

### 🛠️ Organizer Side
- 📊 Crowd monitoring dashboard
- 🚨 Bottleneck detection
- 📢 Broadcast announcements
- ⚡ Emergency alert system

---

## 🧠 Problem It Solves

- ❌ Long waiting times  
- ❌ Crowd congestion & confusion  
- ❌ Poor navigation inside venues  
- ❌ Lack of real-time coordination  

✅ **CrowdPulse solves all with live, data-driven insights**

---

## 🏗️ Tech Stack

- ⚛️ **Frontend:** Next.js, React, TypeScript, Tailwind CSS  
- 🔥 **Backend:** Firebase (Auth + Firestore)  
- 🔄 **Real-Time:** Firestore listeners  
- 🗺️ **Maps:** Google Maps / Mapbox  
- 📊 **Charts:** Recharts  
- 🚀 **Deployment:** Vercel
